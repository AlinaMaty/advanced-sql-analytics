\# Advanced SQL Analytics for E-commerce



\## Описание проекта

Продвинутая SQL-аналитика интернет-магазина:

\- Кохортный анализ

\- Retention по месяцам

\- LTV пользователей

\- Топ-продукты и категории

\- Оконные функции и агрегаты



\## Инструменты

\- SQL (PostgreSQL / SQLite)

\- Python (опционально, для визуализации)



\## Метрики

\- Retention rate

\- LTV по когорте

\- Выручка по категориям

\- Топ-продукты



\## Выводы

\- Наиболее ценные когорты пользователей

\- Продукты с наибольшей выручкой

\- Сезонность выручки

\- LTV позволяет прогнозировать доход



\## Структура проекта

\- data/ - CSV файлы

\- sql/ - SQL-запросы

\- notebooks/ - Python Notebook (опционально)
